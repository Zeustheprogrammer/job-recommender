About us


We developed an end to end web application capable of displaying the current job openings based on the skillset of the users. The users and their information are stored securely in our database and protected from data breaches. The user can interact with the chatbot and can get recommendations based on his/her skills. We used adzuna job search api to get the current job openings in the market which will fetch the data directly from the webpage.
